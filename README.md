Proyecto Final Data Warehouse - Sistema de Correos y Paquetería
Descripción
Este proyecto consiste en el diseño e implementación de un Data Warehouse para el análisis de información de un sistema de correos y paquetería.
El proyecto fue desarrollado utilizando SQL Server, Integration Services (SSIS) y Power BI.
Tecnologías utilizadas
SQL Server
SQL Server Integration Services (SSIS)
Power BI Desktop
GitHub
Componentes del proyecto
Base de Datos Operacional
Base de datos transaccional utilizada para registrar usuarios, sucursales, paquetes y seguimientos.
Data Warehouse
Modelo dimensional compuesto por:
DimUsuario
DimSucursal
DimTiempo
DimEstadoSeguimiento
FactPaquetes
Proceso ETL
Implementado mediante SSIS para extraer, transformar y cargar los datos desde la base operacional hacia el Data Warehouse.
Dashboard
Dashboard desarrollado en Power BI para el análisis de:
Total de paquetes
Promedio de tiempo de entrega
Paquetes por usuario
Paquetes por sucursal destino
